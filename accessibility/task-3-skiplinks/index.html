Assignment 3: Operable_Keyboard Navigation

For people operating web-pages or software interfaces with the keyboard or assistive devices depending on the keyboard interface, mechanisms to bypass blocks on a webpage are very important for efficient use. In the code example below it is your task to implement a “skip link” menu, that allows people to directly jump to the content, menu and footer of the page.

Tasks: Create a skip link menu with the following links:
  1. Jump to content.
  2. Jump to menu.
  3. Jump to footer.
Use the anchor element for the menu and for the content as targets of the skip links. Further,
  1. Make the skip manue invisible (CSS).
  2. Enhance accessibility by adding access-keys to the skip links.
  3. Extra: Enhance focus visibility (CSS).

Enrich the HTML sample below in terms of these accessibility issues! Use moodle to upload your modified and again zipped sourcefile(s).
