Example 1: Accessible Table

The table below is not accessible - it is your task to make it accessible by:

  1. Identifying the cells which are row and column headers.
  2. At the identified cells, using the table header element (th) to replace the table data element (td) to define the headers of rows and columns.
  3. Use scope="row" or scope="col" inside the <th> element (e.g. <th scope="col"/>) to make clear if it is a row or column header.
